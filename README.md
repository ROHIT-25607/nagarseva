# NagarSeva 🚦
### A Citizen-Driven Civic Issue Reporting Platform

> "See a problem? Report it in 30 seconds."

NagarSeva is a web platform where citizens can report civic issues like potholes, broken streetlights, garbage overflow, and water leakage — with photos, location, community upvoting, and live status tracking.

Built as a **Web Technology (WT) Project** for academic submission — Progress 1.

## 📋 Pages
| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Landing page with hero, categories, how it works |
| Login | `login.html` | User login form |
| Register | `register.html` | Sign up with city and locality |
| Dashboard | `dashboard.html` | Stats, recent reports, nearby issues |
| Report Issue | `report.html` | Photo upload, category, location, submit |
| Browse Issues | `feed.html` | All issues with live search and filter |
| Issue Detail | `issue-detail.html` | Full thread, upvote, comments, status timeline |
| Map | `map.html` | Issues plotted on map |
| Leaderboard | `leaderboard.html` | Top contributors by points |
| Profile | `profile.html` | User reports and badges |
| Admin Panel | `admin.html` | Manage and update issue statuses |

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Fonts:** Google Fonts (Sora + DM Sans)
- **Maps:** Google Maps JavaScript API
- **Backend (Progress 2):** Node.js, Express.js, MongoDB
- **Hosting:** Vercel (frontend), Render (backend), MongoDB Atlas (database)

---


## 🚀 How to Run Locally
1. Download or clone this repository
2. Open the folder in **VS Code**
3. Install the **Live Server** extension
4. Right click `index.html` → **Open with Live Server**
5. Site opens in your browser at `localhost:5500`
